# Calculadora em C

Projeto simples desenvolvido em linguagem C com o objetivo de praticar lógica de programação e operações matemáticas básicas. 
O programa funciona via terminal e permite realizar soma, subtração, multiplicação e divisão.

## Instalação e Pré-requisitos

### Pré-requisitos
- Compilador C (GCC)
- Terminal ou Prompt de Comando

### Instalação

Clone o repositório:
```bash
git clone https://github.com/LeticiaVitor/calculadora-c.git

Entre na pasta do projeto:
cd nome-do-repo

Compile o programa:
gcc calculadora.c -o calculadora

## Uso e Exemplos de Comando

Após a compilação, execute o programa com:
```bash
./calculadora
